CTF • OTW

Before you start Link to heading

ssh bandit0@bandit.labs.overthewire.org -p 2220

Login password for Bandit Level 0:
bandit0

Solution / Commands Link to heading

Bash

bandit0@bandit:~$ ls
bandit0@bandit:~$ cat readme

Process Breakdown Link to heading

Step 1 Link to heading 

ls

Once login, we need to first find out what files is in the directory. ls command lists out all the non-hidden files in a directory. We see that only readme exists.

In Windows or Linux OS, files can be hidden by purposely introducing certain characters. Use ls -la command to lists out all files, including non-hidden files in a directory and the corresponding permissions.

Step 2 Link to heading 

cat readme

cat command displays the content in specified file. Use cat readme to display the content in the file named readme. The readme file happens to contain the login password for Bandit Level 1.
