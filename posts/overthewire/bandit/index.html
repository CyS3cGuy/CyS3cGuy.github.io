Bandit Walkthroughs

Difficulty scale:

This wargame is beginner friendly and focuses on basic Linux command-line skills and is a great starting point for beginners.

Each level can be accessed via Secure Shell (SSH) to the corresponding terminal. To SSH, fire up Terminal (Unix) or Command Prompt (Windows). Type in the command ssh bandit<level>@bandit.labs.overthewire.org -p 2220. Then input the password/flag obtained when prompted.

For example, to access Bandit Level 3, use ssh bandit3@bandit.labs.overthewire.org -p 2220.

Note: You are highly recommended to view the provided information in each level page at Bandit OTW before starting each level.

Disclaimer: The login password is rotated on a regular basis. Although the login password in this site may not be relevant anymore, the process to get the next level login password will STILL be relevant. So don’t cheat your way by using the password straight without trying out first!

Glossary friendly: SSH, Terminal
