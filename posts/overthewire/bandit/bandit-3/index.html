Process Breakdown Link to heading

Step 1 Link to heading 

ls

Once login, we need to first find out what files is in the directory. ls command shows that only “spaces in this filename” file exists.

Step 2 Link to heading 

cat spaces\ in\ this\ filename

If we use cat spaces in this filename command, since a whitespace " " is a special character that treats each word as a file itself, it will throw errors as below:

cat: spaces: No such file or directory
cat: in: No such file or directory
cat: this: No such file or directory
cat: filename: No such file or directory

To view the content of the file in which the filename contains whitespace, we can use several methods, either:

  • cat spaces\ in\ this\ filename
  • cat "spaces in this filename"
  • cat 'spaces in this filename'
  • cat ./spaces\ in\ this\ filename

The forward slash \ serves as a way to escape the whitespace so that the whitespace in “spaces in this filename” is treated as part of the string itself.
